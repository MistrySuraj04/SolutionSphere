# Question 9622: Number Of Bathrooms And Bedrooms

**Platform:** StrataScratch

**Difficulty:** Easy

**Problem Statement**

Find the average number of bathrooms and bedrooms for each cityâ€™s property types.
Output the result along with the city name and the property type.

### Solution Query

SELECT
    CITY AS city,
    PROPERTY_TYPE AS property_type,
    AVG(BATHROOMS) AS n_bathrooms_avg,
    AVG(bedrooms) AS n_bedrooms_avg
FROM airbnb_search_details
GROUP BY CITY, PROPERTY_TYPE;

### Explanation

- **Purpose**: Calculates the average number of bathrooms and bedrooms for each property type in each city from the Airbnb search details.
- **Selected Columns**:
  - `CITY AS city`: Selects the `CITY` column and renames it to `city`.
  - `PROPERTY_TYPE AS property_type`: Selects the `PROPERTY_TYPE` column and renames it to `property_type`.
  - `AVG(BATHROOMS) AS n_bathrooms_avg`: Computes the average number of bathrooms and labels it as `n_bathrooms_avg`.
  - `AVG(bedrooms) AS n_bedrooms_avg`: Computes the average number of bedrooms and labels it as `n_bedrooms_avg`.
- **Data Source**: The data is pulled from the `airbnb_search_details` table.
- **Grouping**: Results are grouped by both `CITY` and `PROPERTY_TYPE`, allowing averages to be calculated for each combination of city and property type.
